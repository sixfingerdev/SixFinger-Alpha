API Documentation

Complete guide to integrating SixFinger API

Getting Started

Welcome to the SixFinger API documentation. This API provides powerful AI capabilities for research, code generation, analysis, and more.

Authentication

All API requests require authentication using an API key. Include your API key in the request header:

X-API-Key: your_api_key_here

Alternatively, you can pass it as a query parameter:

?api_key=your_api_key_here

Base URL

https://sixfinger.dev/api/v1

Endpoints

POST /query

Send a general query to the AI.

Request Body

{
    "prompt": "Your question or task here",
    "stream": false
}

Response

{
    "success": true,
    "response": "AI generated response",
    "usage": {
        "user": "username",
        "plan": "free",
        "response_time": 0.5
    }
}

Example

curl -X POST https://sixfinger.dev/api/v1/query \
  -H "X-API-Key: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{"prompt": "Explain quantum computing"}'

POST /research

Conduct comprehensive research on a topic.

Request Body

{
    "topic": "Topic to research"
}

Example

curl -X POST https://sixfinger.dev/api/v1/research \
  -H "X-API-Key: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{"topic": "Latest AI developments"}'

POST /code

Generate code based on requirements.

Request Body

{
    "requirements": "Code requirements description"
}

Example

curl -X POST https://sixfinger.dev/api/v1/code \
  -H "X-API-Key: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{"requirements": "Create a Python function to sort a list"}'

POST /analyze

Analyze provided content.

Request Body

{
    "content": "Content to analyze"
}

GET /usage

Get your API usage statistics.

Response

{
    "user": "username",
    "plan": "pro",
    "usage": {
        "daily": 50,
        "monthly": 1500
    },
    "limits": {
        "daily": 10000,
        "monthly": 250000
    },
    "remaining": {
        "daily": 9950,
        "monthly": 248500
    }
}

Rate Limits

Plan Daily Limit Monthly Limit
Free 100 1,000
Starter 1,000 25,000
Pro 10,000 250,000
Enterprise Unlimited Unlimited

Error Codes

  • 401 Unauthorized - Invalid or missing API key
  • 403 Forbidden - Account inactive or insufficient permissions
  • 429 Too Many Requests - Rate limit exceeded
  • 400 Bad Request - Invalid request parameters
  • 500 Internal Server Error - Server error
